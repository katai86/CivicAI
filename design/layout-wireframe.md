# Layout Wireframe Description

## Desktop (3-column)
- **Left Panel**: user identity, rank, XP bar, badges, CTA
- **Center**: full-bleed map canvas with subtle glow and animated markers
- **Right Panel**: live feed + mini cards + solved preview

## Mobile
- **Top**: minimal brand bar
- **Center**: full map
- **Bottom**: 5-item nav with glowing FAB in the middle
- **Report modal**: slide-up sheet with category icons + XP reward

## Landing Hero
- Fullscreen city-night aesthetic
- Big headline + CTA, center aligned
- Background: faint animated map glow
