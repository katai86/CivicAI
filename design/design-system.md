# KÖZ.TÉR – Design System

## Typography
- Font: Montserrat (Regular 400, SemiBold 600, ExtraBold 800, Black 900)
- Headline (H1): 36–44px, Black
- H2/H3: 22–28px, ExtraBold/SemiBold
- Body: 14–16px, Regular
- Buttons: 14–16px, ExtraBold

## Color System (Dark mode first)
- Background: `#1E2A38`
- Deep gradient: `#1E2A38 → #0F1721`
- Accent Green: `#00C48C`
- Accent Orange: `#FF7A00`
- Alert Red: `#FF3B3B`
- XP Purple: `#8E44FF`
- Text: `#E6EEF7`
- Muted: `#9FB0C3`

## Glassmorphism
- Card bg: `rgba(22,33,45,0.7)`
- Border: `rgba(255,255,255,0.08)`
- Blur: `backdrop-filter: blur(10px)`
- Radius: `12px`
- Shadow: `0 8px 24px rgba(0,0,0,0.35)`

## Spacing
- Scale: 8 / 12 / 16 / 20 / 24 / 32 / 40

## Buttons
- Primary CTA: Accent Orange + glow
- Secondary: Glass card style

## Iconography
- Simple, geometric, two-tone
- Use 18–20px for inline icons

## Motion
- Ease: `cubic-bezier(0.2,0.6,0.2,1)`
- Duration: 150–250ms for microinteractions
- Pulse: 2–3s for active markers
